荔链直达 - 专业农产品溯源平台
============================

🌐 系统概述：
-----------
模仿主流溯源平台，采用 webTrace/trace.html?id=编码格式
使用Base64编码技术，保护溯源码安全性
区块链技术确保数据不可篡改

📁 核心文件：
-----------
专业溯源演示.html      - 主演示页面（推荐使用）
webTrace/trace.html     - 溯源查询页面
妃子笑荔枝_专业二维码.png - 妃子笑荔枝专业二维码
桂味荔枝_专业二维码.png   - 桂味荔枝专业二维码

-----------

🎯 使用流程：
-----------
1. 打开 "专业溯源演示.html" 或 "测试跳转.html"
2. 扫描产品二维码（推荐使用通用二维码）
3. 自动跳转到专业溯源页面
4. 查看完整的种植→加工→物流→验证信息

🚨 如果扫描后显示URL文本而不是跳转：
-----------
1. 尝试使用"通用二维码"版本
2. 如果不行，尝试"直接跳转二维码"
3. 或者手动点击页面上的"手动查看溯源"链接

🔍 溯源信息包含：
-----------
✅ 产品基本信息（名称、编码、产地、农户）
🌱 种植全过程时间线
🏭 加工处理详细记录
🚚 物流运输追踪信息
🔒 区块链验证证书

🛠️ 技术特色：
-----------
- ✅ Base64编码保护溯源码
- ✅ 区块链哈希验证
- ✅ 响应式设计，适配手机
- ✅ 无需网络，完全离线工作
- ✅ 专业UI设计，类似主流平台

📱 测试方法：
-----------

1. 打开 "专业溯源演示.html"
2. 用手机扫描二维码

方法2 - 直接访问：
1. 在浏览器中输入URL测试
2. webTrace/trace.html?id=TEMyMDI0MDExNzAwMQ==

方法3 - 手动输入：
1. 打开 webTrace/trace.html
