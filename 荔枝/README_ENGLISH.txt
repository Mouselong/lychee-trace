LITCHI TRACE - Blockchain Traceability System
========================================

Installation and Usage Guide
----------------------------------------

STEP 1: Install Environment
----------------------------------------
1. Install Node.js
   - Download: https://nodejs.org/dist/v18.18.2/node-v18.18.2-x64.msi
   - Double-click to install, make sure to check "Add to PATH"
   - Restart computer after installation

2. Install MongoDB
   - Download: https://www.mongodb.com/try/download/community
   - Select: Windows -> x64 -> msi
   - Choose "Complete" installation type
   - Check "Install MongoDB as a Service"

STEP 2: Run Installation Script
----------------------------------------
Double-click to run:
- install_guide.bat (Recommended) - Complete installation guide
- OR check_installation.bat - Environment check and dependencies

STEP 3: Start System
----------------------------------------
Double-click to run:
- run_system.bat - Start frontend and backend services

STEP 4: Access System
----------------------------------------
Open browser and visit:
http://localhost:3000

========================================
Troubleshooting:
1. Make sure Node.js is installed and added to PATH
2. Make sure MongoDB service is running
3. Check network connection
